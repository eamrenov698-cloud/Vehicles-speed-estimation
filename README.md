# Оценка скорости транспортных средств с использованием YOLO Ultralytics

Этот проект предназначен для **оценки скорости транспортных средств** на видео с использованием предобученной модели **YOLO** от Ultralytics. Модель позволяет детектировать транспортные средства и приблизительно оценивать их скорость без необходимости иметь собственный датасет или обучать модель с нуля.

---

## Особенности

- Использует предобученную модель YOLO Ultralytics для обнаружения автомобилей, грузовиков и других транспортных средств.
- Реализована базовая оценка скорости на основе перемещения объектов между кадрами видео.
- Простая интеграция и запуск на локальном компьютере или сервере.
- Не требует собственного датасета и обучения модели с нуля.
